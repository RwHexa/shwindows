03 · So einfach geht's
Eine Datei. Ein Doppelklick. Fertig.
Kein Installations-Assistent. Keine Registry-Einträge. Keine Admin-Rechte nötig.
1
📥
Datei kopieren
SmartHome-Steuerung.exe auf den Desktop oder in einen beliebigen Ordner kopieren.
2
▶️
Doppelklick starten
Anwendung öffnet sich sofort. Zugangsdaten zur HiveMQ-Cloud sind bereits konfiguriert.
3
🏠
Sofort steuern
Lampen schalten, Heizung regeln, Status sehen – innerhalb von 30 Sekunden einsatzbereit.

05 / 08
04 · Funktionsumfang
🔌
Geräte schalten
Shelly Gen3 (Steckdosen, Lampen)
  • EIN/AUS per Knopfdruck
  • Echtzeit-Verbrauch (W, V, A, kWh)
  • Helligkeit + Farbtemperatur dimmen
  • Live-Status-Anzeige
🌡️
Sensoren auslesen
ESP32 mit DHT11 / PIR
  • Temperatur in °C
  • Luftfeuchte in %
  • Bewegungserkennung
  • Helligkeit / Dämmerung (LDR)
🪟
Rollladen steuern
Pro Raum einzeln
  • Position 0-100% per Slider
  • Visuelle Echtzeit-Darstellung
  • Mehrere Rollos parallel
🚨
Alarm + Push
Bewegungs- und Test-Alarm
  • PIR-Bewegungsmelder ESP32
  • Push aufs Smartphone (ntfy.sh)
  • Bis zu 3× Wiederholung
  • Quittierung per Knopfdruck
🗑️
Müll-Termine
Automatische Erinnerung
  • Heute und morgen farbig markiert
  • Restmüll, Bio, Gelb, Papier, Schadstoff
  • Stündliche Aktualisierung
  • Aus eigener Termin-Datei
🏠
Grundriss-Ansicht
3D-Visualisierung
  • Eigener Wohnungsgrundriss
  • Klickbare Lampen pro Raum
  • Thermostat im Raum
  • Übersichts- und Detailansicht
